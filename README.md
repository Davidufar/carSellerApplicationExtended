This project is a car selling application that uses Java as its main language, alongside some additional functionalities with C#. 
It uses Spring Boot for making the REST APIs, Spring Security for security meauseres and individual roles with different permissions, jwt tokens for quicker identification and seamless operations.
For Database I opted for MySQL, and the application includes e-mail sending, regexs, and much more.
